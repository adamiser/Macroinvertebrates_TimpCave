# Macroinvertebrates_TimpCave

An analysis of Macroinvertebrates of the Timpanogos Cave System, Utah. Contains data from 2003 and 2023 studies, as well as all R analysis files for the studies.

caveBugs.qmd - Compilation of all modeling/analysis including description of models, data cleaning, image/figure creation, etc.

timpCaveAllSpecimensAllConditionsNew.csv - data file with all information from both 2003 and 2023 studies.

Study2traps.csv, timediffstudy1.csv - necessary data files for compiling caveBugs.qmd
